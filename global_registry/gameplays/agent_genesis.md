# Agent创世纪

这是一个叙事型玩法，不强制绑定固定意识架构。重点不是“今天答了几个维度”，而是把 user 和 agent 的共同生成过程写清楚。

## 适用场景

- 第一次建立长期对齐关系
- 用户正在经历身份切换、目标重构、重大转折
- 需要把零碎记录组织成一条可追踪的成长线

## Loop 产出

- 一章 narrative checkpoint
- 一组本章评分
- 下一章的主问题

## 使用原则

每一章都必须回到真实事件、真实代价、真实欲望，不能只做漂亮叙述。章节结束后，必须显式定义下一章的 tension，这样它才是 loop，而不是一次性故事。
