# 情侣对齐挑战

这是一个双人玩法。重点不是让双方“看起来更和谐”，而是尽快找出关系中最贵的误判点。

## 适用场景

- 关系进入重复争执
- 一方觉得“你根本不懂我”
- 双方都愿意做结构化记录，而不是只在冲突当下表达

## Loop 产出

- 双方的关系 snapshot
- 一张差异 scorecard
- 下一周唯一的 repair action
